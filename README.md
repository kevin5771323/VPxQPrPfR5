# 二手交易管理系统 java1215

## 项目概述

二手交易管理系统是基于B/S架构和MVC模式开发的前后端分离系统。本项目使用Java技术栈，结合SpringBoot框架和Mybatis持久层框架，前端使用Vue.js，支持多平台运行，满足不同操作系统的需求。

## 技术栈

- **开发语言**：Java
- **前端框架**：Vue.js
- **后端框架**：SpringBoot
- **数据库**：Mysql
- **持久层框架**：Mybatis
- **开发工具**：IDEA、JDK1.8、Maven

## 功能模块

### 前台功能

- **首页**：全部查询、分类查询、关键字搜索
- **发布物品**：用户可以发布二手物品
- **查看消息**：查看系统通知和私信
- **个人中心**：编辑个人信息、编辑收货地址
- **状态查询**：
  - 我发布的
  - 我下架的
  - 我收藏的
  - 我卖出的
  - 我买到的

### 后台功能

- **交易物品管理**：
  - 上线的二手物品（可以违规下架）
  - 下架的二手物品（可以永久删除）
- **订单信息管理**：删除订单信息
- **用户信息管理**：
  - 正常用户（可封号）
  - 违规用户（可以解封）
  - 管理员（可添加管理员）

## 系统角色

- 普通用户：可以发布物品、查看消息、管理个人物品和交易状态等。
- 管理员：负责用户管理、交易物品管理和订单信息管理。

## 运行环境

- **操作系统**：Windows/Mac
- **Web 服务器**：内置Tomcat

项目名称/

## 核心亮点

- **前后端分离**：系统采用前后端分离的设计，提高了开发效率和系统的维护性。
- **跨平台支持**：支持Windows和Mac操作系统，满足不同用户的运行需求。
- **易于部署**：提供详细的部署教程，方便用户快速上线运行。
- **丰富的功能**：覆盖了二手交易的各个环节，从前台的用户交互到后台的管理，提供了完整的解决方案。
